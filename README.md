# 🏠 AutoCAD 2D House Plan

### 📐 Residential Floor Plan Designed Using AutoCAD

A professionally drafted **2D residential house plan** created using **AutoCAD**. This project demonstrates architectural drafting techniques, accurate dimensioning, room planning, and engineering drawing standards. It reflects my learning and practical experience in Computer-Aided Design (CAD) as a Civil Engineering student.

---

## 📌 Project Overview

This project contains a **2D House Plan** created using **AutoCAD** as part of my learning journey in engineering drawing and computer-aided design (CAD). The drawing demonstrates the basic principles of residential planning, including room layout, wall dimensions, doors, windows, and other architectural elements.

The objective of this project is to improve my AutoCAD drafting skills and understand the standards used in creating professional 2D architectural drawings.
